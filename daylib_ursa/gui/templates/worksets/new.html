Quick Start Guide

A workset is a batch of analysis inputs processed together. Use this page to capture the operational metadata, then attach either a saved manifest or a TSV you upload here.

  1. Name the workset and select the pipeline metadata.
  2. Choose a saved manifest or upload a stage_samples.tsv file.
  3. Set priority, notifications, and cluster preferences.
  4. Submit once the cost estimate and sample preview look right.
